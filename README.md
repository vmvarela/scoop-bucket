# scoop-bucket

Scoop bucket for [vmvarela](https://github.com/vmvarela) tools.

## Usage

```powershell
scoop bucket add vmvarela https://github.com/vmvarela/scoop-bucket
scoop install sql-pipe
```

## Packages

| Package | Description |
|---------|-------------|
| [sql-pipe](https://github.com/vmvarela/sql-pipe) | Read CSV from stdin, query with SQL, write CSV to stdout |
